Koffeine's ESLint shareable config

## Installation

```sh
npm install --save-dev @koffeine/eslint-config eslint
```

## Usage

Add the following to your `eslint.config.js` file:

```js
import koffeine from '@koffeine/eslint-config';

export default [
    ...koffeine
];
```
